## NG Commands

### New Project

```sh
$ ng new PROJECT-NAME
```

### Run Project

```sh
# Default
$ ng serve

# Personalized
$ ng serve --host 0.0.0.0 --port 4201
```

Navigate to `http://localhost:<PORT>/`

### New Component

```sh
$ ng generate component my-new-component

# Or
$ ng g component my-new-component
```
